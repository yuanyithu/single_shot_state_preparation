# exp18_q001_left_combined_summary

- Purpose: combine and compare q=0.0100 left-window scans after exp13/14 suggested the useful threshold region is not p>0.24.
- Main sources: exp15 + exp16 pooled dense grid, plus exp17 fine grid and earlier exp13/14 references.
- Main result: pooled dense data places the L3-L4 sign change near p≈0.223, while L4-L5 is still slightly negative at p=0.230; the robust window is therefore around p≈0.22-0.235, not the far-right p>0.24 region.
